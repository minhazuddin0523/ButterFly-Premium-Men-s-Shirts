# ButterFly - Premium Men's Shirts 🦋
https://butterfly-shirts.vercel.app/

ButterFly is a premium e-commerce platform for men's shirts, built with a focus on clean UI and seamless user experience.

## Features
- **Dynamic Product Listing:** Integrated with Supabase/Database.
- **Search & Filter:** Advanced filtering by size, color, and price.
- **Admin Panel:** Easy product management for admins.
- **Secure Payment:** Integrated payment gateway for safe transactions.
- **Responsive Design:** Fully optimized for all devices.

## Tech Stack
- **Frontend:** React.js / Vite
- **Styling:** Tailwind CSS
- **Backend/Database:** Supabase / Firebase
- **Deployment:** Vercel
